# 🔐 SecureVault — Менеджер паролей с AMPG v1

**Безопасный менеджер паролей для Android с авторским алгоритмом мнемонической генерации.**

Проект разработан в рамках магистерской диссертации.

---

## 🎯 Цель проекта

Создать безопасное приложение для локального хранения паролей с двумя режимами генерации:
- **Обычный** — криптостойкие случайные пароли
- **Мнемонический (AMPG v1)** — запоминающиеся пароли по пользовательской фразе

---

## ✨ Основные функции

### 🔑 Генерация паролей
- **Обычный генератор**: SecureRandom, без повторяющихся символов
- **Мнемонический генератор AMPG v1**: детерминированная генерация по фразе
- **5 вариантов** в каждом наборе с разными стратегиями
- **Кнопка "Ещё варианты"** для перегенерации

### 🔄 Ротация паролей
- Индивидуальная ротация с напоминаниями
- Массовая ротация с предпросмотром
- **Перемешивание паролей между сервисами** внутри профиля
- Автоматическое обновление `nextRotationDate`

### 📜 История паролей
- Зашифрованное хранение старых паролей
- Просмотр только после подтверждения мастер-пароля
- Типы операций: random / mnemonic / manual / shuffled / swapped
- Связанные сервисы и записи

### 🛡️ Безопасность
- AES-256-GCM через Android Keystore
- PBKDF2 для мастер-пароля (100,000 итераций)
- HMAC-SHA256 для fingerprint паролей
- Constant-time compare
- BruteForceGuard с блокировкой
- Очистка charArray после использования

### 📤 Импорт/Экспорт
- CSV с предупреждениями о безопасности
- Корректная обработка quoted полей
- Генерация новых UUID при импорте
- Поддержка разных профилей

### 🔍 Аудит безопасности
- Поиск слабых паролей
- Проверка просроченной ротации
- Обнаружение повторного использования паролей
- Проверка 60% уникальности

### 📱 QR-коды
- Генерация защищённых QR для записей
- QR НЕ содержит пароль
- Привязка к устройству и профилю
- Срок действия 24 часа
