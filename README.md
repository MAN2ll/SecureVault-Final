# SecureVault

Безопасный менеджер паролей для Android с многоуровневой защитой и уникальными функциями.

## Основные возможности

### Безопасность
- AES-256-GCM шифрование паролей
- Мастер-пароль с PBKDF2 (100000 итераций)
- Профили с PIN-кодами
- Защита от brute-force
- Удаление данных после N неудачных попыток

### AMPG v2 (Adaptive Mnemonic Password Generator)
AMPG v2 создаёт запоминаемый пароль из пользовательской фразы.

**Режимы:**
- **Обычный режим** — пароль для одного пользователя
- **Режим двух пользователей** — один пароль на две равные части

**Особенности:**
- Длина в режиме двух пользователей: 16, 18 или 20 символов
- Части равные: 16 = 8+8, 18 = 9+9, 20 = 10+10
- По умолчанию без разделителя
- Emoji не используются
- Сервис и дата ротации могут участвовать в генерации
- Повторы символов не допускаются (проверка без учёта регистра)
- Генерация детерминированная: одинаковые параметры дают одинаковый пароль

**Пример:**
Фраза: "метроном жёлтый камень"
Сервис: Gmail
Дата ротации: 07.2026
Результат: Metr0n!7Yel0w$9


### Ротация паролей
- 4 режима генерации: случайный, AMPG v2, ручной, из существующих
- Циклическая перестановка между сервисами
- Управляемая перекрёстная ротация с выбором донора
- Защита от повторов в истории
- Журнал всех изменений
- Сортировка и фильтрация

### QR-коды
- "Вечные" QR-коды (работают после ротации)
- Привязка к устройству и профилю
- Сохранение в галерею
- Сканирование с расшифровкой через мастер-пароль

### Backup
- CSV экспорт/импорт (совместимость)
- Полный защищённый backup v3 (профили + пароли)
- Шифрование AES-256-GCM
- Ключ из пароля через PBKDF2 (200000 итераций)
- 3 режима импорта: новые/объединить/пропустить
- Перенос между устройствами (plaintext пароли в зашифрованном backup)
- История паролей переносится с fingerprint

### Уведомления
- Фоновая проверка раз в день (WorkManager)
- Уведомления о просроченных паролях
- Уведомления о скоро истекающих (7 дней)
- Запрос разрешения на Android 13+

### Аудит
- Поверхностный аудит (без расшифровки)
- Глубокий аудит (с расшифровкой через мастер-пароль)
- Поиск перекрёстных дубликатов
- Проверка уникальности символов
- Проверка отличия от предыдущего пароля (≥60%)

## Технические решения

- **Room Database** с шифрованием
- **Android Keystore** для ключей HMAC
- **WorkManager** для фоновых задач
- **Jetpack Compose** для UI
- **Hilt** для dependency injection
- **MVVM** архитектура

## Установка

1. Клонируйте репозиторий
2. Откройте в Android Studio
3. Соберите проект (Build → Make Project)
4. Установите на устройство

## Требования

- Android 7.0 (API 24) или выше
- Разрешение на камеру (для QR)
- Разрешение на уведомления (Android 13+)
